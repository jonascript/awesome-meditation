# Awesome Meditation
A list of meditation resources to learn how to meditate that are online and free.

- [Palouse MBSR](https://palousemindfulness.com/)
- [Isha Kriya - Free Online Guided Meditation](http://www.ishafoundation.org/Ishakriya)
- [28 Day Mindfulness Challenge](https://mindfulnessexercises.com/free-online-mindfulness-courses/28-day-mindfulness-challenge/)
- [Buddhism Through the Scriptures Course, Harvard](https://www.edx.org/course/buddhism-through-its-scriptures)
- [Search Inside Yourself](https://mindfulnessexercises.com/free-online-mindfulness-courses/search-inside-yourself-program-developed-at-google/)
- [Coursera, Demystifying Mindfulness](https://www.coursera.org/learn/mindfulness)
